What is this?

There is a new way to use Probable Futures maps: as embeds. Map embeds are like mini versions of Probable Futures maps that can be added to any online page, report, post or story. It works with both the public maps and Probable Futures Pro.

How it works

Go to any Probable Futures map, position the map as you would like it, and click the embed icon on the right side. The icon looks like this: iframe

On the public maps, the button is hidden. Use this link to show the button.

These embeds can be used to show any Probable Futures map in any warming scenario focused on any place in the world. They can also include additional or custom data by using Probable Futures Pro.
